Map Explorer

What is Map Explorer?

Map Explorer is freely available tool to explore Line-of-Sight (LOS) on maps for the D&D Miniatures Skirmish game.

Running Map Explorer

You need a Java 5 Runtime Environment to run Map Explorer. One way to check this is to enter

    java -version

on the command line. It should print something like this:

    java version "1.5.0_04"
    Java(TM) 2 Runtime Environment, Standard Edition (build 1.5.0_04-b05)
    Java HotSpot(TM) Client VM (build 1.5.0_04-b05, mixed mode, sharing)

The version number in the first line of the output must at least be 1.5.xxx. If it is 1.3.xxx or 1.4.xxx you need a more recent Java version.

Download the current version of Map Explorer: MapExplorer-20051201. (Version history)

Unzip the archive. It will create a Directory MapExplorer containing the program and the maps.

On Windows Systems, you can start Map Explorer by double-clicking MapExplorer.bat.

On UNIX or Linux, user the mapexplorer shell script.
